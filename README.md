# Personal Finance Tracker
---

A full-stack Personal Finance Dashboard built using the MERN stack. It enables users to securely manage income and expenses, track transactions, and visualize financial data through interactive charts and analytics.

---

## 🚀 Features

-  Secure User Authentication (JWT)
-  Add, Edit, and Delete Income
-  Add, Edit, and Delete Expenses
-  Dashboard with financial overview
-  Charts and Analytics
-  Transaction History
-  Responsive User Interface
-  Search and Filter Transactions

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JSON Web Token (JWT)
